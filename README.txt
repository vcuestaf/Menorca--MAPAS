
INSTRUCCIONES PARA PUBLICAR TUS MAPAS EN GITHUB PAGES

1. Entra en https://github.com y crea una cuenta gratuita (si aún no tienes una).
2. Crea un nuevo repositorio, por ejemplo: "menorca-mapas".
3. Haz clic en 'Add file' > 'Upload files' y sube todo el contenido de esta carpeta:
   - Dia1_Menorca.html
   - Dia2_Menorca.html
   - Dia3_Menorca.html
   - Dia4_Menorca.html
   - index.html

4. Una vez subido todo, ve a:
   - Settings > Pages (barra lateral)
   - En "Source", selecciona: Deploy from a branch → Branch: main, Folder: / (root)
   - Haz clic en "Save".

5. Después de unos segundos, GitHub generará un enlace como este:
   https://TU_USUARIO.github.io/menorca-mapas/

Desde ese enlace podrás abrir el `index.html` y acceder a cada mapa desde tu móvil.

¡Disfruta tus vacaciones en Menorca!
